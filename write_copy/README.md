# 'write' copy 

> 

## system calls used
* `getlogin()`
  - get the username of the logged in user 
* `ttyname()`
  - get the name of the terminal(e.g. /dev/pts/4) 
* `gethostname()`
  - get the host name 

## macros
* `STDIN_FILENO`
  - file descriptor for stdin 
* `UTMP_FILE`
  - file descriptor for utmp
* `O_RDONLY`
  - open file should be opened in read only mode
